〇 アプリケーションの概要（なにができるシステムなのか等）
	このアプリはゲームに関するサイトを音声認識から表示してくれるサイトである。
	ゲームの情報が載っているサイト,ゲームクリエイターの経歴などゲームに興味のある人が
	一度は読んでみたい,もしくは読んでおいて損はないというサイトである。
〇 簡単な使用方法（なんとしゃべればいいのか等）
	マイクをオンにしたら,「ゲームの情報」「ゲームジャンル」「任天堂」と
	話せば,それぞれに該当するサイトのURLが表示されるというシステムである。
	また,「宮本茂」「桜井政博」「坂口博信」と話せば,各ゲームクリエイターに関する
	情報が載ったサイトのURLが表示されるという仕組みである。
	ちなみに,「こんにちは」や「お疲れ様です」といった日常の挨拶にも返事を返せるようになっている。
〇 工夫した点（注目してほしいアピールポイント等）
	なんといってもゲームに関する情報を集約的に載せたことである。
	レストランの場所や天気などを表示するサイトは多々あるが,ゲームに
	関する情報を表示するサイトはあまりない。しかし,ゲーム業界は
	常に新しい作品が生まれ,評価されている。それを踏まえ,今回は
	ゲームに関係する情報を表示するサイトとした。
	
	また,技術面での工夫として音声認識システムという事から,他の検索サイトとは異なり,ユーザーとの
	コミュニケーションを取れることも遊び心として実装した。ゆえにJSONのライブラリと
	スクリプトに入力した文字列双方に反応する仕組みにプログラムしたことが技術面でのアピールポイントである。
